#  2.2不要被阶乘吓倒
1、给定一个整数N，那么N的阶乘N！末尾有多少个0呢？例如：N=10，N！=3628800，N！的末尾有两个0.

思路：
1.1、有哪些数字相乘可以得出10，2和5相乘，由于2出现的频率低于5出现的频率，以此统计5的个数即可N！= K * 10（M次方），且K不能被10整出。

int count = 0;
for (int i = 0; i <= N; i++) {
     int j = i;
     while (j % 5) {
         count++;
          j /= 5; 
     }
}
